Detailed Description

Recursive descent parser for the Scheme programming language.

The Parser class implements a recursive descent parser that converts a stream of tokens into an Abstract Syntax Tree (AST). The parser processes a wide range of Scheme expressions including:

Core Features:

  • Atoms (numbers, identifiers, booleans)
  • S-expressions (function calls)
  • Special forms (if, lambda, define)
  • Lists and vectors
  • Quoted expressions

Advanced Features:

  • Macro system (define-syntax, syntax-rules)
  • Let expressions
  • Proper tail calls
  • Import statements

Example usage:

auto scanner = std::make_shared<Scanner>();
auto parser = std::make_shared<Parser>();
parser->initialize(scanner);
std::vector<Token> tokens = scanner->tokenize("(define x 42)");
parser->load(tokens);
auto ast = parser->parse();
